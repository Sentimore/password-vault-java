# Password Vault (Java)

Байтуганов Захар, группа МПБ-301
Краткое описание: консольный менеджер паролей, который генерирует сложные пароли и хранит их в зашифрованном виде под одним мастер-паролем.

## Стек

- Язык и фреймворк
  - Java 17 (консольное приложение, без фреймворков)
- Ключевые библиотеки
  - Стандартный JDK: `javax.crypto`, `java.security.SecureRandom`, `java.util.Base64`
- Сервисы и интеграции
  - Нет внешних сервисов. Все данные хранятся локально в файле `vault.dat` на диске.

## Быстрый старт

### Требования
- Установленный Java JDK 17+.
- Git (если проект клонируется с GitHub).

### Установка

git clone https://github.com/Sentimore/password-vault-java.git
cd password-vault-java
javac PasswordVault.java

### Запуск

java PasswordVault

Далее программа предложит выбрать режим:
- создать зашифрованный файл с записями (site / user / pass),
- либо расшифровать и показать уже сохранённые записи по мастер-паролю.

## Функциональность

- Генерация случайных паролей заданной длины с выбором: заглавные буквы, цифры, спецсимволы.
- Хранение записей в виде блоков:
  - `site: ...`
  - `user: ...`
  - `pass: ...`
- Шифрование всех записей одним мастер-паролем (PBKDF2 + AES/CBC + IV).
- При добавлении новых записей старые расшифровываются, дополняются новыми и шифруются снова.
